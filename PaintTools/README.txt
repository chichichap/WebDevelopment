1. Instructions on how to use the application
The application is pretty much self-explanatory. 
- the Paste button will paste the copied shape at the same location (you can move it afterwards)
- select the shape then click the resize button to resize the shape

2. About the application structure
- Shape subclasses: Lines, Triangles, and Rectangles
- Virtual Functions: draw(), testHit()
- Javascript library for color picking:bgrins.github.io/spectrum
- Three mouse events: mouse down, mouse move, mouse up
- jQuery used to get and detect changes of: line_width, line_color, and fill_color
